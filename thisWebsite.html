This Website

 I got the foundational HTML/CSS template for this website from HTML5 UP and personalized it to create the unique design you see now! Making this portfolio website has been so much fun and has reminded me how much I ❤ to code!

Changes Made

 While platforms like Weebly and Wix facilitate easy website creation, I wanted to make this website uniquely mine. I crafted this site using HTML, CSS, and JavaScript, which provided me with a greater degree of customization. To build this website, I used a foundational template from HTML5 UP, then made major alterations like adjusting colors, incorporating personalized images, and updating personal information. I also dedicated (too much) time to subtle changes aimed at enhancing the user experience. For instance, I added a scroll-to-top button in the bottom right corner and ensured that the "read less" button smoothly scrolls back to the content to prevent users from losing their place. In the individual project pages, I incorporated a button to return to the project page, and upon doing so, it returns users to the section they were originally on. Additionally, I made links open in a new tab to maintain the user's position and included a button for downloading my resume. These are all small details that greatly improve the user experience.
